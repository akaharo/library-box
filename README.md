# Library Box

Esse site é uma página inicial para uma biblioteca escolar.

A ideia é mostrar alguns livros do acervo, avisos da biblioteca, uma área de favoritos e uma opção simples para reservar uma leitura.

## Arquivos

- `index.html`
- `styles.css`
- `script.js`
- `background.mp4`
- `logo-library-box.png`
- `logo-library-box-icon.png`

## Como abrir

É só abrir o arquivo `index.html` no navegador.

## Funções da página

- menu responsivo;
- filtros de livros;
- botão de favorito;
- prateleira interativa;
- reserva de livro;
- vídeo de fundo;
- animações e transições suaves.
